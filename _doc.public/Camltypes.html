Module Camltypes

module Camltypes: sig .. end

The camel types in CamlBattle

type move_t =
| Attack
| Defense
| Heal

The abstract value representing move type

type element_t = string

The type of value representing the element type

type hp = int

The type of the health points

type moves = move_t list

The type of move identifiers that a player has

The abstract type of values that represents either the left, middle, or right box in the blind box mini-game

type blind_boxes =
| LeftBox
| MiddleBox
| RightBox

The abstract type of values that represents either a good ending where the player recieves a boost to their HP, a bad ending where a player loses HP points, or a neutral end where nothing happens

type endings =
| GoodEnd
| BadEnd
| NeutralEnd

The abstract type of values representing the effectiveness of the power of one element against another

type effect =
| Weak
| Normal
| SuperEffect 
type t = {
   hp : int;
   level : int;
   element_t : element_t;
   moves : move_t list;
   exp : int;
   defense : int;
   current_move : int;
}

The abstract type of values representing camels and enemies. current move is denoted by 0,1,2,3. 0 is at the beginning of the game. 1 is attack, 2 is heal, and 3 is defense

type stage = string

The abstract type of values representing a stage.

exception UnknownStage of stage

Raised when an unknown biome type is encountered

exception UnknownElement of element_t

Raised when an unknown element type is encountered

val exp_update : t -> t

exp_update t checks to see if experience is at or above 100 after battle. If it is, will return new caml object with level increased by 1. Requires t is a valid camltype t

val effect_multi : effect -> float

effect_multi effect returns the multiplier of the result of the match up between enemy and player caml. Requires: effect is a valid type effect

val attack_multi : element_t -> element_t -> effect

attack_multi x returns how effective the combination of the attacks are in the form of (element_t * element_t) where the first element is attacking. Requires x is a valid tuple containing two valid element_t types.

val current_hp : t -> hp

current_hp t returns the current hp of the character in battle. Requires: valid type t

val caml_type : t -> element_t

caml_type caml returns the elemental type of the caml

val effect_match : t -> t -> element_t * element_t

effect_match t t returns the elemental type of both the caml player and the enemy in a tuple

val updated_hp : t -> int -> t

updated_hp t int returns a caml with the updated hp after having taken damage.

val updated_defense : t -> int -> t

updated defense t int returns a caml with the updated hp after calling a defensive move

val current_defense : t -> int

current_defense t returns the current defense of the character in battle

val attack_stage_multi : element_t -> stage -> effect

attack_stage_multi caml stage returns the attack multiplier of the caml according to what stage biome they are in.
